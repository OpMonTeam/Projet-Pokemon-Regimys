Comment jouer ?   --------VERSION 0.07--------------

Sur Linux : Après vous être assuré d'avoir une machine vituelle java 1.8 sur votre ordinateur, cliquez sur ExecLinux.sh

Sur Windows: Après avoir installé Oracle Java 1.8, cliquez sur Exec.bat

Sur Mac (ou sur un autre système si la methode plus haut ne fonctionne pas) : ouvrez le terminal dans le dossier oû est stocké Regimys.jar et tapez dans le terminal java -jar Regimys.jar

Questions et Aide

*Quand j'execute ExecLinux.sh ou que je tape la commande, rien ne se passe, pourquoi ?

Verifier bien que vous ouvrez un terminal DANS LE DOSSIER DU JEU, et qu'une machine virtuelle java est installé, si vous ne savez pas ce que c'est et que vous voulez juste jouer au jeu, taper d'abord avant de lancer le jeu : 

sudo apt-get install openjdk-8-jre 

puis ensuite      

java -jar Regimys.jar
