Comment jouer ?  
 --------VERSION 0.07--------------


Sur Linux : Après vous être assuré d'avoir une machine vituelle java 1.8 sur votre ordinateur, cliquez sur ExecLinux.sh


Sur Windows: Après avoir installé Oracle Java 1.8, cliquez sur Exec.bat


Sur Mac (ou sur un autre système si la methode plus haut ne fonctionne pas) : ouvrez le terminal dans le dossier ou est stocké Regimys.jar et tapez dans le terminal java -jar Regimys.jar
