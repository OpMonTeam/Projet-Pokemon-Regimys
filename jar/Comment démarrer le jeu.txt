Comment jouer ?   --------VERSION 0.05.2--------------

Sur Linux : Après avoir télécharger OpenJDK Runtime, cliquer sur ExecLinux.sh

Sur Windows: Après avoir télécharger Java, cliquer sur Exec.bat

Sur Mac (ou sur un autre système si la methode plus haut ne fonctionne pas) : ouvrer le terminal dans le dossier oû est stocker Regimys.jar et taper dans le terminal java -jar Regimys.jar
